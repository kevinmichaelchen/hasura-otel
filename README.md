# hasura-otel

Exploring Hasura's latest OpenTelemetry (OTel) features.

## Getting started

### Prerequisites

To get started, install [pkgx][pkgx] with:

```shell
curl -fsS https://pkgx.sh | sh
```

> [!NOTE]
> You can easily uninstall pkgx with `sudo rm $(which pkgx)` and `sudo rm -rf ~/.pkgx`

### Running everything

```shell
pkgx task start
```

[pkgx]: https://pkgx.sh/
